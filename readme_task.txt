Разместить у себя на хостинге форму HTML с полем для проверки ИНН физического лица на наличие признака недостоверности. Детали задачи ниже.

Необходимо написать класс на PHP с именем Authenticity в файле authenticity.php, который имеет публичный метод get($inn), на вход из HTML-формы принимает ИНН физического лица и возвращает результат в виде массива:

Объявление: public function get ($inn)
Входной параметр: $inn - ИНН физического лица
Результат - один из массивов:
[
   'inn' => $inn
   ,'message' => 'По заданным критериям поиска сведений не найдено.'
   ,'authenticity' => true
]
или
[
   'inn' => $inn
   ,'message' => 'Наличие признака недостоверности'
   ,'authenticity' => false
];
Адрес для проверки наличия записи о недостоверности: https://pb.nalog.ru/search.html

Проверка тестового задания:

Файл для проверки результатов тестового задания - разместить у себя на хостинге (например index.php):

_здесь_вставить_форму_для_проверки_ИНН_на_достоверность_
<?php
if (isset($_POST['inn'])){
    include_once ('authenticity.php');
    $authenticity = new Authenticity();
    echo '<p>Проверка ИНН=' . $_POST['inn'] . ':</p><pre>';
    print_r($authenticity->get());
    echo '</pre>';
}

Пример ИНН с ограничением 784100362403, 332501373163, без ограничения - 253402065152 или 772739580300
